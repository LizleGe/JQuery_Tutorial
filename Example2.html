Example 2: Explanation

This animation that is generated by JQuery is called the fade effect. Once a button is clicked, the animations will appear and then fade into view.

In this Example 2, as shown in box 3 below, tree elements in HTML (div1, div2 and div3), will fade in different speeds, once the fade button is clicked. Its also called a 'fading-in' effect.

This is a great interactive element that can be used in your website to make it more interactive and draw users to your website. A refresh button is also added to repeat the fade action animation.

Example 2: The Code


                    
                        
                        $(document).ready(function(){
                        $("button.fade").click(function(){
                            $("#div1").fadeIn();
                            $("#div2").fadeIn("slow");
                            $("#div3").fadeIn(3000);
                        });
                        
                        $("button.refresh").click(function(){
                            location.reload();
                        });
                    });
