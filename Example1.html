Example 1: Explanation

A Ticker tape animation is generated using JQuery or JavaScripts. In example 1 on this page, the ticker tape animation is demonstrated and the code is shown. This type of animation is content that is continuously scrolling or updating content.

The content is scrolling in a "tape" or "row" format accross the webpage. This type of animation is advantageous to advertise certian products or will be used to display imporant information such as a news update. In conclusion, the information is continiuously scrolling horizontally across your sreen displaying information.

How it is implemented into your code, is by created a 'div' tag, in which you create the contact you want to display in your 'tape'. Therafter, you can creat a CSS style to design and control the easthetics of your ticker tape animation.

The Jquery code as shown below in example 2, will give the animation its interactivity. You can therfore control the speed of the information scrolling or create a start and stop button as show, in example 3 below.

Example 1: The Code

 
                $(document).ready(function() {
                  let isRunning = false;
                  const tickerContent = $(".ticker-content");
                  
                  
                  function toggleTicker() {
                    if (!isRunning) {
                      tickerContent.css("animation-play-state", "running");
                      $("#startStopButton").text("Stop");
                    } else {
                      tickerContent.css("animation-play-state", "paused");
                      $("#startStopButton").text("Start");
                    }
                    isRunning = !isRunning;
                  }
                  
                  
                  $("#startStopButton").click(toggleTicker);
                });
