Example 4: Explanation

This JQuery feuture is also used to add an interactive experience for the user.

Once the fade button is clicked, the sentence, the element, will fade or dissappear. It will use the class element 'hide' that will dissappeat the element. The element reffered to in this example is the sentence. This is illustrated in the third box below. A refresh button is added, that will refresh the page and therfore alow you to see the hidden element (the sentence) once again.

This is a great feature to use to enhance the interactive experience of the user on a website.

Example 4: The Code


                    
                        
                    $(document).ready(function(){
                        $("#hide_button").click(function(){
                            $(".hide").hide();
                        });
                        $("#show").click(function(){
                            $(".hide").show();
                        });
                        });
