Example 3: Explanation

This animation that is generated by JQuery is called the 'Start-Animation' effect.

once the button is clicked as shown in box below, the animation will slide to the right.

How it works, is when the button is clicked, it will calculate the maximum position to the right of the red boder box. Thereafter, it uses the class, animation-block in order for the element(the light green block), to move from its original position to the right side of the red border box.

In short, the light green block will move smoothly from left to right within the red border box, once the start button is clicked. This JQuery feature is great to use in your website to make it more interactive for the user.

A refresh button is also added to reload the webpage to repeat the animation. This is done via the location.reload().

Example 3: The Code


                    
                        
                    $(document).ready(function(){
                        $("button.start_animation").click(function(){                        
                        var maxRightPosition = $(".box3_example3").width() - $(".animation-block").width();                                        
                        $(".animation-block").animate({left: maxRightPosition + 'px'});
                        });
                        $("button.refresh").click(function(){
                        location.reload();
                        });
                        });
